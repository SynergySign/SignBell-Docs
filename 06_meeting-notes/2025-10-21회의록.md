## 논의 내용
- **(HTTPS 설정)**: 송민재 님이 프론트엔드와 백엔드의 HTTPS 설정을 완료했음을 공유했습니다. (개발팀 공통: Chocolatey 설치 후 인증키를 받아야 개발 진행 가능)
- **(API 및 문서 작업)**: 강관주 님이 REST API 문서 작성을 완료했으며, 백승현 님 담당 부분은 백엔드 작업 완료 후 업데이트하기로 했습니다. 고동현 님은 사용자 플로우, ERD(논리/개념), WebSocket API 명세서 작성을 완료했습니다.
- **(UI 및 기능 개발)**: 신동준 님이 실시간 수어 퀴즈 UI를 완료했으며, 백승현 님은 FastAPI를 프론트엔드에 연결하는 작업을 진행 중임을 공유했습니다.

## 결정 사항
- 실시간 수어 퀴즈 UI 및 HTTPS 설정이 완료됨에 따라, **강관주 님과 고동현 님이 실시간 수어 퀴즈 프론트엔드 개발**을 시작하기로 결정했습니다.
- 백승현 님은 완료된 HTTPS 설정을 기반으로 **FastAPI 연결 문제를 해결**하기로 했습니다.
- 송민재 님은 **약관 동의 관련 프론트엔드 작업을 시작**하고, 페이지 이동 시 로그아웃되는 버그를 수정하기로 했습니다.
- 신동준 님은 **수어 개인 학습 페이지 UI 개발**을 진행하기로 했습니다.
