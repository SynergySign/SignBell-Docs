### 2. MVP 기능 명세 논의
- 서비스의 핵심 기능을 정의하고, 각 기능의 세부 동작 방식을 구체화했습니다.

- **MUST (필수 구현 기능)**
    - **로그인**: 카카오 SSO를 통한 소셜 로그인
    - **AI 모델**: 자체 수집 데이터를 활용한 수어 인식 모델 학습
    - **개인 수어 학습**:
        - API를 통해 제공되는 단어 리스트 및 설명 영상 제공
        - '거울 모드'를 통해 사용자가 자신의 수어 동작을 실시간으로 확인하며 연습 (사람 상체 모양 프레임 고정)
          - '거울 모드'로 촬영한 영상데이터는 좌표를 추출하여 좌표 추출 데이터만 DB에 저장 후 AI 학습 데이터 용도로 사용 (약관동의 필요)
    - **영상통화 퀴즈 (WebRTC 기반)**:
        - **참여 인원**: 최소 2명 ~ 최대 4명
        - **방 기능**: Public 방 생성, 방 리스트 조회, 방 입장/퇴장 (카메라 없는 사용자는 입장 제한), 방장만 시작 버튼 누르기
        - **퀴즈 진행 방식**:
            1. 랜덤 단어 출제
            2. 선착순으로 버튼을 누른 사용자가 문제 풀이 기회 획득
            3. 문제 풀이 사용자의 화면이 커지고, 3초 카운트다운 후 5초 내에 수어 동작 수행 (사람 상체 모양 프레임 고정)
            4. 5초 후 MediaPipe로 좌표를 추출하여 AI 모델로 유사도 검사 및 정답/실패 처리
            5. 정답 시 +100점, 실패 시 -50점 / 방 내에서만 유효한 순위별 차등 점수 적용
            6. 차등 점수 1번쨰 시도자 100점 지급 다음 시도자부터는 90점,80점,70점
            7. 정답자가 나오거나 모든 인원이 실패하면 다음 문제로 진행
            8. 모든 퀴즈가 끝나면 점수별로 1등,2등,3등 표시
            9. 퀴즈가 끝나면 다시 방으로 이동 (재시작 or 방 나가기 가능)

- **SHOULD (추후 구현 고려 기능)**

    - **개인 수어 학습**
        - **개인 학습 퀴즈**: 개인 학습용 랜덤 단어 퀴즈, 튜토리얼, 캠을 이용한 연습(유사도 체크) 기능
        - **부가 기능**: 모의고사, 오답노트 기능 추가

- **UI/UX**
    - 전체적인 레이아웃은 양옆에 여백을 두는 웹 앱 형태로 구성
