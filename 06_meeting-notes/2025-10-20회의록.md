## 논의 내용
### **1. 팀원별 진행 상황 공유**
* **고동현:**
  * 백엔드 수어 퀴즈 기능 개발 완료 (예외 처리 포함).
  * 프론트엔드 연동 작업 진행 예정.
* **백승현:**
  * FastAPI 모듈화 진행 (개인 수어, 퀴즈 엔드포인트 통합).
  * 프론트엔드 연동을 위한 테스트 페이지 작업 중.
* **강관주:**
  * 게임 목록 API 변경 완료 (진행 중/대기 중 상태 모두 조회 가능하도록 수정).
  * 방 번호 검색 API 추가 완료.
* **신동준:**
  * 메인 페이지 UI 구현 완료.
  * 실시간 수어 퀴즈 UI 금일 오후까지 완료 예정.
* **송민재:**
  * AWS 서버 배포 준비 중 (EC2, S3 활용).
  * 쿠버네티스를 이용한 배포 준비.
  * 배포 전, 프론트엔드 로그인/약관 동의(axios 설정), 헤더 로그인 유지 기능 우선 작업 예정.

### **2. 주요 기능 및 인프라 논의**
* **개인 수어 학습 페이지:** 페이지네이션 대신 **무한 스크롤** 방식으로 구현하기로 결정.
* **개인 학습 데이터 테이블:** 유저 학습 이력 관리를 위한 신규 테이블 생성 건은 우선 **보류**.
* **EC2 인스턴스:** 용량(RAM 8GB, 스토리지 30GB + 추가 50GB)에 대한 **예상 비용 확인 필요**.
* **EC2 폴더 구조:** 배포 시 혼동을 피하기 위해 FastAPI 레포지토리 이름 변경 필요.
