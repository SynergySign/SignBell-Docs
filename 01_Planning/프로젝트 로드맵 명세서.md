# SignBell - 프로젝트 로드맵

본 문서는 SignBell 서비스의 현재 MVP 단계를 넘어, 사용자 경험을 개선하고 커뮤니티를 활성화하며, 지속 가능한 플랫폼으로 성장하기 위한 중장기 발전 계획을 정의합니다.

* **작성자:** [고동현](https://github.com/rhehdgus8831)
* **작성일**: 2025-10-27
* **최종 수정일**: 2025-10-27
* **문서 버전**: v1.0.0

**대상 독자:**

* **기획자 / PM**: 프로젝트의 장기적인 비전과 단계별 목표를 이해하고 로드맵 관리
* **백엔드 개발자**: 향후 개발될 기능의 기술적 요구사항 및 시스템 확장 방향 예측
* **프론트엔드 개발자**: 사용자 인터페이스 및 경험의 진화 방향 예측
* **디자이너 (UX/UI)**: 서비스 발전 방향에 따른 디자인 시스템 확장 및 개선 계획 수립
* **DevOps / 인프라 엔지니어**: 서비스 성장에 따른 인프라 확장 및 운영 계획 수립 참고
* **QA / 테스트 엔지니어**: 로드맵 기반 테스트 전략 및 계획 수립
* **AI 모델러**: 수어 인식 모델 개선 및 신규 AI 기능 개발 방향 참고
* **신규 합류자**: 프로젝트의 비전과 향후 발전 계획을 빠르게 파악

---

## 1. 개요

본 문서는 SignBell 서비스의 초기 MVP 출시 이후, 지속적인 성장과 발전을 위한 단계별 로드맵을 제시합니다. 각 단계별 목표와 주요 기능/개선 사항을 정의하여 프로젝트의 방향성을 명확히 하고, 팀 전체가 공동의 목표를 향해 나아갈 수 있도록 합니다.

---

## 2. 최종 목표

**"AI 기반 실시간 수어 학습을 넘어, 농인과 청인이 함께 소통하고 배우는 포용적인 커뮤니티 플랫폼으로 성장한다."**

SignBell은 단순한 수어 학습 도구를 넘어, 수어를 통한 소통의 장벽을 낮추고, 농인과 청인이 자연스럽게 교류하며 서로를 이해하는 사회적 가치를 실현하는 플랫폼을 지향합니다.

---

## 3. 단계별 로드맵

### 3.1 Phase 1: MVP - 핵심 기능 구현 및 시장성 검증 (현재)

* **목표**: 서비스의 핵심 가치인 'AI 기반 실시간 수어 학습 및 퀴즈 게임'을 구현하고, 초기 사용자 반응을 통해 시장성을 검증합니다.
* **주요 기능**:
    * **사용자 인증**: 카카오/구글 소셜 로그인, 약관 동의 시스템
    * **개인 수어 학습**: 한국정보화진흥원 수어 데이터 기반 학습 콘텐츠 제공
    * **실시간 퀴즈 게임**:
        - 게임방 생성/입장/퇴장
        - WebSocket 기반 실시간 통신
        - AI 수어 인식을 통한 정답 판정
        - 라운드별 점수 집계 및 랭킹
    * **기본 프로필 관리**: 닉네임, 프로필 이미지, 누적 점수 조회
    * **AI 수어 인식**: FastAPI 기반 실시간 수어 동작 인식 (기본 단어)
    * **WebRTC 영상 통신**: Janus Gateway를 통한 실시간 영상 공유

---

### 3.2 Phase 2: 서비스 고도화 및 사용자 경험 개선 (v2.0)

* **목표**: MVP 운영 피드백을 반영하여 사용 편의성을 개선하고, 서비스 안정성 및 신뢰도를 높이는 기능을 추가합니다.
* **주요 기능 / 개선**:
    * **알림 시스템 도입**:
        - 게임 초대 알림
        - 친구 요청 알림
        - 학습 목표 달성 알림
        - 이벤트 및 공지사항 알림 (Push, In-app)
    * **AI 수어 인식 개선**:
        - 인식 가능 수어 단어 확장
        - 인식 정확도 향상 (학습 데이터 확대)
        - 복합 동작 인식 지원
    * **수어 전문가 협업 시스템**:
        - 수어 전문가 영상 데이터 수집 플랫폼 구축
        - 전문가 검증을 거친 양질의 수어 영상 데이터베이스 확보
        - 전문가 기여에 대한 보상 시스템 (포인트, 인증서 등)
        - 사용자 제출 수어 영상 전문가 검증 시스템
    * **사용자 편의 기능**:
        - 수어 학습 가이드 및 튜토리얼
        - 수어 검색 기능 개선 (카테고리별, 난이도별)
        - 학습 진도 추적 및 복습 알림
    * **UI/UX 개선**:
        - 사용자 피드백 기반 디자인 개선
        - 반응형 웹 최적화
        - 다크 모드 지원
    * **관리 기능 강화**:
        - 부적절한 닉네임/프로필 신고 시스템
        - 게임방 신고 및 제재 시스템
        - 관리자 대시보드 (사용자/게임 통계)
    * **모바일 앱 개발**: PWA 방식으로 모바일 최적화

---

### 3.3 Phase 3: 핵심 기능 확장 및 커뮤니티 강화 (v3.0)

* **목표**: 서비스의 핵심 기능을 확장하고, 사용자 간 상호작용을 촉진하여 활성 커뮤니티를 구축합니다.
* **주요 기능 / 개선**:
    * **다양한 게임 모드 추가**:
        - 1:1 대결 모드
        - 팀 대항전 모드
        - 타임 어택 모드
        - 테마별 퀴즈 (일상, 직장, 학교 등)
    * **학습 콘텐츠 심화**:
        - 문장 단위 수어 학습
        - 상황별 수어 회화 학습
        - 수어 문법 학습
        - 사용자 제작 학습 콘텐츠 업로드
        - 전문가 제작 프리미엄 학습 콘텐츠 (정확한 동작, 상세 설명)
    * **랭킹 시스템**:
        - 전체/주간/월간 랭킹
        - 카테고리별 랭킹
        - 친구 랭킹
    * **뱃지 및 업적 시스템**:
        - 학습 완료 뱃지
        - 게임 승리 뱃지
        - 연속 출석 뱃지
        - 특별 업적 (첫 승리, 100게임 참여 등)
    * **친구 시스템**:
        - 친구 추가/삭제
        - 친구 초대 (게임방)
        - 친구 활동 피드
    * **실시간 채팅**:
        - 게임방 내 채팅
        - 1:1 채팅
        - 그룹 채팅
    * **사용자 맞춤 추천**:
        - AI 기반 학습 콘텐츠 추천
        - 실력에 맞는 게임방 추천
        - 관심사 기반 친구 추천

---

### 3.4 Phase 4: 플랫폼 확장 및 생태계 구축 (v4.0)

* **목표**: 서비스 범위를 확장하고, 다양한 파트너십 및 수익 모델을 통해 지속 가능한 플랫폼 생태계를 구축합니다.
* **주요 기능 및 계획**:
    * **서비스 범위 확장**:
        - 다국어 지원 (영어, 일본어, 중국어)
        - 국제 수어 학습 콘텐츠 추가
        - 해외 사용자 대상 서비스 확장
    * **전문가 연계 서비스**:
        - 수어 전문가 Q&A
        - 1:1 수어 멘토링 매칭
        - 수어 통역사 연결 서비스
        - 전문가 네트워크 구축 및 협업 플랫폼
        - 전문가 인증 시스템 (자격증, 경력 검증)
        - 전문가 제작 콘텐츠 수익 분배 모델
    * **오프라인 연계**:
        - 수어 학습 모임 주선
        - 오프라인 수어 대회 개최
        - 농인 커뮤니티 연계 이벤트
    * **교육 기관 연계**:
        - 학교/기업 대상 수어 교육 프로그램 제공
        - 수어 교육 커리큘럼 개발
        - 수어 능력 인증 시스템
    * **데이터 기반 개인화**:
        - 학습 패턴 분석 기반 맞춤형 학습 경로 제공
        - 게임 실력 분석 및 개선 제안
        - 사용자 행동 데이터 기반 서비스 최적화
    * **Open API 제공**:
        - 수어 인식 API 제공
        - 수어 학습 콘텐츠 API 제공
        - 외부 서비스 연동 지원
    * **수익 모델 다각화**:
        - **프리미엄 구독**:
            - 광고 제거
            - 고급 학습 콘텐츠 접근
            - 무제한 게임 참여
            - 전문가 멘토링 할인
        - **B2B 솔루션**:
            - 기업/학교 대상 수어 교육 플랫폼 제공
            - 수어 인식 API 라이선스 판매
        - **브랜드 굿즈**:
            - SignBell 캐릭터 상품
            - 수어 학습 교재 및 교구
            - 수어 관련 문화 상품
