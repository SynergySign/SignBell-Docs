## 기능 우선순위 (MoSCoW)

SignBell 프로젝트의 MVP(최소 기능 제품) 구현을 위한 핵심 기능과 향후 확장 계획은 다음과 같습니다.

### 1. Must Have (MVP - 필수 구현 기능)

현재 프로젝트 기간 내에 **반드시 구현해야 할 핵심 기능**입니다.

*   **로그인**: 카카오 SSO를 통한 소셜 로그인
*   **AI 모델**: 자체 수집 데이터를 활용한 수어 인식 모델 학습 및 서빙
*   **개인 수어 학습**:
    *   API를 통해 제공되는 단어 리스트 및 설명 영상 제공
    *   '거울 모드'를 통해 사용자가 자신의 수어 동작을 실시간으로 확인하며 연습 (사람 상체 모양 프레임 고정)
    *   '거울 모드'로 촬영한 영상데이터는 좌표를 추출하여 좌표 추출 데이터만 DB에 저장 후 AI 학습 데이터 용도로 사용 (약관동의 필요)
*   **영상통화 퀴즈 (WebRTC 기반)**:
    *   **참여 인원**: 최소 2명 ~ 최대 4명
    *   **방 기능**: Public 방 생성, 방 리스트 조회, 방 입장/퇴장 (카메라 없는 사용자는 입장 제한), 방장만 시작 버튼 누르기, 방 참여 인원은 게임 준비 가능 (방장은 기본 값 TRUE)
    *   **퀴즈 진행 방식**:
        1.  랜덤 단어 출제
        2.  선착순으로 버튼을 누른 사용자가 문제 풀이 기회 획득
        3.  문제 풀이 사용자의 화면이 커지고, 3초 카운트다운 후 5초 내에 수어 동작 수행 (사람 상체 모양 프레임 고정)
        4.  5초 후 MediaPipe로 좌표를 추출하여 AI 모델로 유사도 검사 및 정답/실패 처리
        5.  정답 시 +100점, 실패 시 -50점 / 방 내에서만 유효한 순위별 차등 점수 적용 (1번째 시도자 100점, 다음 시도자부터 90점, 80점, 70점)
        6.  정답자가 나오거나 모든 인원이 실패하면 다음 문제로 진행
        7.  모든 퀴즈가 끝나면 점수별로 1등, 2등, 3등 표시
        8.  퀴즈가 끝나면 다시 방으로 이동 (재시작 or 방 나가기 가능)

### 2. Should Have (차기 업데이트 - 우선적으로 고려할 기능)

MVP 구현 후 **우선적으로 추가할 가치가 있는 기능**입니다.

*   **개인 학습 확장 기능**:
    *   **개인 학습 퀴즈**: 랜덤 단어 퀴즈 모드
    *   **튜토리얼**: 수어 기본 동작 연습 가이드 제공
    *   **유사도 체크**: 캠으로 연습 시 AI 피드백 제공 (버튼 클릭 → 3초 카운트다운 → 5초 동작 → 유사도 결과)
    *   **모의고사**: 여러 단어를 연속으로 테스트하는 기능
    *   **오답노트**: 틀린 단어를 기록하고 복습할 수 있는 기능
*   **기타 확장 기능**:
    *   **학습 진도율 추적**: 사용자의 학습 진행 상황을 시각적으로 보여주는 기능
    *   **일일/주간 학습 통계**: 학습량, 정확도 등의 개인 통계를 제공
    *   **UI/UX 개선**: 전체적인 레이아웃은 양옆에 여백을 두는 웹 앱 형태로 구성

### 3. Could Have (중장기 확장 - 장기적으로 고려할 기능)

향후 프로젝트 발전 방향에 따라 **추가할 수 있는 아이디어성 기능**입니다.

*   **수어 사전 기능**: 단어 검색 및 해당 수어 영상 제공
*   **커뮤니티 기능**: 학습자 간 소통 게시판, 팁 공유, 질문 답변 게시판
*   **다국어 지원**: 다양한 언어의 수어 학습 지원 (예: 미국 수어, 일본 수어)
*   **수익 모델 기능**:
    *   광고 삽입
    *   프리미엄 학습 콘텐츠 제공 (구독제)
    *   AI 피드백 상세화 (무료 vs 구독자 심화 피드백)
*   **모바일 앱 출시**: 웹 앱을 넘어 네이티브 모바일 앱 개발
*   **AI 챗봇**: 수어 관련 질문에 답변하거나 학습 가이드를 제공하는 대화형 챗봇
*   **개인화된 학습 경로 추천**: 학습 이력 및 성과 기반으로 다음 학습 콘텐츠 추천
*   **배지 및 업적 시스템**: 학습 동기 부여를 위한 게이미피케이션 요소
*   **친구 초대 및 함께 학습 기능**: 친구를 초대하여 학습하거나 퀴즈를 즐기는 기능

### 4. Won’t Have (초기 제외 - 현재 고려하지 않는 기능)

현재 프로젝트 범위에서는 **고려하지 않는 기능**입니다.

*   고급 분석 대시보드 (사용자 통계, 장기 리포트 등 복잡한 데이터 분석 기능)
*   멀티 플랫폼 동시 출시 (데스크탑, 웨어러블 등 웹 앱 외의 플랫폼)
*   상세 커뮤니티 기능 (오픈 채팅, 그룹 스터디 관리 등 고도화된 소셜 기능)
*   외부 캘린더 연동 등 수어 학습과 직접적인 관련이 없는 기능
