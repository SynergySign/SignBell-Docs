## 1. 개요 (Overview)

SignBell은 **AI 기반 실시간 수어 학습 플랫폼**입니다. 한국수어(KSL) 학습의 접근성을 높이고, 청인과 농인 사이의 소통 장벽을 허무는 것을 목표로 합니다. 수어 개인 학습과 AI 기술을 통한 WebRTC 기반의 실시간 퀴즈를 통해 효과적이고 즐거운 수어 학습 경험을 제공하는 데 중점을 둡니다.

---

## 2. 목적 (Purpose)

* 수어 단어 영상과 거울 모드를 통해 사용자가 혼자서도 정확한 수어 동작을 연습할 수 있도록 합니다.
* WebRTC 기반 실시간 퀴즈에서 AI 피드백을 통해 학습 효과를 측정하고 다른 학습자와 함께 즐기며 배우는 게이미피케이션 경험을 제공합니다.
* 개인 학습 거울 모드에서 수집된 학습 데이터를 활용하여 AI 모델이 지속적으로 성능을 개선하는 선순환 구조를 구축합니다.

---

## 3. 핵심 기능 (Core Features)

* **소셜 로그인**: 카카오톡 SSO를 통해 간편하게 회원가입 및 로그인 지원
* **개인 수어 학습**: API 기반 단어 리스트 및 설명 영상 제공, 거울 모드를 통한 실시간 연습
* **실시간 퀴즈**: WebRTC 기반 영상통화 방에서 2~4명이 함께 참여하는 게이미피케이션 퀴즈
* **AI 기반 피드백**: 실시간 퀴즈에서 MediaPipe 좌표 추출 및 자체 학습 모델을 통한 수어 동작 인식
* **학습 데이터 수집**: 개인 학습 거울 모드에서 좌표 데이터를 수집하여 AI 모델 학습에 활용

## 4. 기능 요구사항 (Functional Requirements)

### 4.1 회원/인증

| ID      | 기능 상세                 | 우선순위   | 비고                                |
|---------|-----------------------|--------|-----------------------------------|
| FR-01   | 카카오톡 SSO 로그인을 통한 회원가입 | MUST   | OAuth 2.0 기반 소셜 로그인               |
| FR-02   | JWT 기반 로그인 및 로그아웃     | MUST   | 액세스/리프레시 토큰 방식, 쿠키에 저장            |
| FR-02-1 | 액세스 토큰 자동 갱신          | MUST   | 리프레시 토큰을 이용한 무중단 인증 유지            |
| FR-03   | 최초 로그인 시 약관 동의        | MUST   | 서비스 이용약관 (필수) 및 학습 데이터 수집 동의 (선택) |
| FR-04   | 프로필 조회                | SHOULD | 닉네임, 프로필 이미지, 퀴즈 점수 조회            |
| FR-05   | 프로필 수정                | SHOULD | 닉네임, 프로필 이미지 수정                   |
| FR-06   | 회원 탈퇴                 | SHOULD | 탈퇴 처리 및 데이터 삭제                    |

* **회원가입**: 카카오톡 소셜 로그인을 통한 간편 가입
* **로그인/로그아웃**: JWT 토큰 기반 인증 시스템
* **토큰 갱신**: 액세스 토큰 만료 시 리프레시 토큰을 이용해 자동으로 재발급하여 사용자 경험 유지
* **약관 동의**: 최초 로그인 시 서비스 이용약관 및 학습 데이터 수집 동의 절차 필수. 학습 데이터 수집 동의는 회원가입의 필수 조건이며, 데이터 제공 모드에서 수집되는 좌표 데이터 활용에 적용됨
* **프로필 관리**: 사용자 정보 조회 및 수정 기능

※ 인증 방식의 기술적 구현 상세(토큰 만료 시간, 저장 위치, 암호화 방식 등)는 [기술 명세서](링크)를 참고하세요.


### 4.2 개인 수어 학습

| ID    | 기능 상세          | 우선순위   | 비고                        |
|-------|----------------|--------|---------------------------|
| FR-07 | 단어 리스트 조회      | MUST   | 학습 가능한 한국수어(KSL) 단어 목록 제공 |
| FR-08 | 단어 설명 영상 재생    | MUST   | 선택한 단어의 수어 동작 영상 재생       |
| FR-09 | 거울 모드 활성화/비활성화 | MUST   | 웹캠을 통한 실시간 미러링 기능         |
| FR-10 | 실시간 화면 분할      | MUST   | 거울 모드 시 영상과 웹캠 동시 표시      |
| FR-11 | 상체 프레임 가이드     | SHOULD | 사용자 동작 가이드라인 표시           |
| FR-12 | 좌표 데이터 추출 및 저장 | MUST   | 거울 모드에서 동작 좌표 수집          |

* **단어 리스트**: 한국수어(KSL) 학습 단어 목록 제공
* **설명 영상**: 각 단어의 정확한 수어 동작 시연 영상
* **거울 모드**:
    - 웹캠을 활성화하여 사용자의 동작을 실시간으로 확인
    - 화면 분할: 설명 영상(상단)과 거울 모드(하단)를 동시에 표시
    - 사용자가 영상을 참고하며 따라할 수 있도록 지원
    - 상체 프레임 가이드로 올바른 포즈 유도
* **데이터 수집**: 데이터 제공 모드 사용 시 동작 좌표를 자동으로 수집하여 AI 모델 학습 데이터로 활용 (회원가입 시 선택 동의)

※ 좌표 추출 방식, 데이터 파이프라인, 저장 형식 등 기술적 상세는 [수어 인식서버(FAST API) 기술명세서](https://github.com/SynergySign/SignBell-Docs/blob/dev/03_Specifications/SignBell%20-%20%EC%88%98%EC%96%B4%EC%9D%B8%EC%8B%9D%EC%84%9C%EB%B2%84%20%EA%B8%B0%EC%88%A0%EB%AA%85%EC%84%B8%EC%84%9C.md)를 참고하세요.


### 4.3 실시간 퀴즈 (WebRTC 기반)

| ID    | 기능 상세             | 우선순위 | 비고                               |
|-------|-------------------|---------|----------------------------------|
| FR-13 | 퀴즈 방 생성           | MUST    | Public 방, 방 제목 설정                |
| FR-14 | 퀴즈 방 리스트 조회       | MUST    | 대기, 진행 중인 방 목록 및 참여 인원 표시        |
| FR-15 | 퀴즈 방 입장           | MUST    | 로그인 한 회원이라면 누구나 입장 가능            |
| FR-16 | 퀴즈 방 대기실          | MUST    | 2~4명 참여 가능, 실시간 영상 연결            |
| FR-17 | 퀴즈 방 참여자 준비 상태 변경 | MUST    | 웹캠을 연결한 참여자만 준비 상태로 변경 가능        |
| FR-18 | 방장 권한             | MUST    | 방 생성자만 퀴즈 시작 가능, 모든 참여자 준비 완료 필수 |
| FR-19 | 퀴즈 방 퇴장           | MUST    | 모든 참여자 퇴장 및 퀴즈 방 삭제              |
| FR-20 | 랜덤 단어 출제          | MUST    | 퀴즈 시작 시 랜덤 문제 제시                 |
| FR-21 | 선착순 응답 시스템        | MUST    | 정답 도전 기회 선착순 배정                  |
| FR-22 | 화면 확대 및 카운트다운     | MUST    | 응답자 화면 확대 및 준비 시간 제공             |
| FR-23 | 수어 동작 수행          | MUST    | 제한 시간 내 수어 동작 수행                 |
| FR-24 | 동작 좌표 추출          | MUST    | 사용자 수어 동작 분석을 위한 좌표 추출           |
| FR-25 | AI 유사도 검사         | MUST    | AI 모델을 통한 정답 판정                  |
| FR-26 | 정답/오답 처리 및 점수 부여  | MUST    | 순위별 차등 점수 시스템                    |
| FR-27 | 다음 문제 진행          | MUST    | 자동 문제 전환                         |
| FR-28 | 최종 순위 표시          | MUST    | 퀴즈 종료 시 순위 집계 및 표시               |
| FR-29 | 대기실로 자동 복귀        | MUST    | 퀴즈 종료 후 대기실 복귀                   |

* **퀴즈 방 구조**:
    - **참여 인원**: 최소 2명 ~ 최대 4명
    - **방 생성**: Public 방, 방 제목 설정 가능
    - **준비 상태 변경**: 카메라 권한 필수, 웹캠 연결 필요
    - **대기실**: 실시간 영상 연결, 모든 참여자 화면 표시
    - **방장 권한**: 퀴즈 시작 권한

* **퀴즈 진행 방식**:
    1. 랜덤 단어 출제
    2. 모든 참여자에게 [정답 도전하기] 버튼 표시
    3. 선착순으로 버튼을 누른 사용자가 문제 풀이 기회 획득
    4. 문제 풀이 사용자의 화면 확대, 다른 참여자 화면은 축소
    5. 카운트다운 후 제한 시간 내 수어 동작 수행
    6. 동작 좌표 추출 및 AI 유사도 검사
    7. 정답/오답 판정 및 점수 부여
    8. 다음 문제로 자동 진행
    9. 모든 퀴즈 종료 시 최종 순위 표시
    10. 대기실로 자동 복귀 (방장은 퀴즈 재시작 가능)

* **점수 시스템**:
    - **정답 시**: 순위별 차등 점수 (1위 100점, 2위 90점, 3위 80점, 4위 70점)
    - **오답 시**: -50점
    - **최종 순위**: 누적 점수 기준

※ WebRTC 연결 방식, 시그널링 서버 구성 등 기술적 상세는 [기술 명세서](링크)를 참고하세요.

## 5. 제약 사항

* **비즈니스 제약**
    - 한국수어(KSL)만 지원 (다른 국가 수어 미지원)
    - MVP 단계에서는 Public 방만 지원 (Private 방 미지원)
    - 실시간 화상 퀴즈만 지원 (녹화 영상 미지원)
    - 개인 학습은 연습 모드만 제공 (AI 실시간 피드백 미제공)
    - AI 피드백은 실시간 퀴즈에서만 제공
    - 결제 기능 미지원
    - 학습 데이터 수집 동의는 회원가입 필수 조건

※ 기술적 제약사항, 성능 요구사항, 보안 정책 등은 [기술 명세서](링크)를 참고하세요.

## 6. 용어 정의 (Glossary)

* **한국수어(KSL)**: Korean Sign Language, 대한민국의 농인 사회에서 사용되는 고유한 수화 언어
* **거울 모드(Mirror Mode)**: 웹캠을 통해 사용자의 동작을 실시간으로 미러링하여 보여주는 학습 모드
* **MediaPipe**: Google이 개발한 오픈소스 프레임워크로, 실시간으로 사람의 포즈, 손, 얼굴 등의 좌표를 추출하는 라이브러리
* **WebRTC**: Web Real-Time Communication, 웹 브라우저 간 실시간 음성 및 영상 통신을 가능하게 하는 기술
* **방장(Host)**: 퀴즈 방을 생성한 사용자로, 퀴즈 시작 권한을 가짐
* **유사도 검사**: AI 모델이 사용자의 수어 동작과 정답 동작의 유사도를 계산하여 정답 여부를 판정하는 과정
* **좌표 데이터**: MediaPipe로 추출한 신체 관절 포인트의 X, Y, Z 좌표 값
* **액세스 토큰**: 사용자 인증 상태를 확인하기 위한 단기 유효 토큰
* **리프레시 토큰**: 액세스 토큰을 재발급하기 위한 장기 유효 토큰
