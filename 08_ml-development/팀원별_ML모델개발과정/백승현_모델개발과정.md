# 수어 인식 모델 개발 과정 요약: CNN-BiLSTM-Attention-PE

[백승현](https://github.com/sirosho) 팀원의 수어 인식 모델 개발 과정은 **데이터 특징 추출 개선**을 기반으로, $\text{CNN-BiLSTM-Attention}$ 구조에 **$\text{Positional Encoding (PE)}$를 추가**하여 **복잡한 수어 동작의 시간 순서 관계**를 명시적으로 학습하는 데 중점을 두었습니다.

---

### 1단계: 초기 모델 및 문제 정의

| 특징 | 내용 | 결과/문제점 |
| :--- | :--- | :--- |
| **초기 아키텍처** | $\text{Bi-LSTM} + \text{Attention}$ | 좌표 노이즈에 취약, $\text{Triplet Loss}$ 학습 불안정. |
| **주요 난제** | 전반적으로 모델의 신뢰도가 떨어짐. 특히 **'미국'**과 **'무한'** 단어 간의 혼동 (손 깍지 vs. 손 교차) 문제 발생. | |

---

### 2단계: 특징 추출 (데이터 전처리) 개선

모델의 안정성과 강건성을 높이기 위해 $\text{MediaPipe}$ 추출 좌표에 대한 특징 공학이 단계적으로 적용되었습니다.

| 특징 | 내용 | 개선 효과 |
| :--- | :--- | :--- |
| **좌표계** | 절대 좌표 $\to$ **계층적 상대 좌표** (어깨/손목 기준) | 사용자 위치, 거리, 각도 변화에 대한 **강건성(Robustness) 극대화**. |
| **차원/깊이** | $\text{2D} \to \mathbf{3\text{D}}$ ($\mathbf{Z}$축) 정보 복원 및 **$\text{Z}$축 감쇠/평활화** | **입체적인 동작 학습 안정화** 및 $\text{Z}$축 노이즈 감소. |
| **노이즈 제거** | $\text{15}$개 $\text{Pose}$ $\to$ $\mathbf{6}$개 $\text{Pose}$ (어깨, 팔꿈치, 손목) | 수어와 무관한 **몸통 잡음(Noise) 제거**. |
| **새로운 특징** | **양손 손목 간 $\text{3D}$ 상대 거리** 특징 추가 | 양손 수어 동작의 **상대적 위치 관계**를 명시적으로 모델에 제공. |
| **$\mathbf{\Delta X}$ (속도) 특징 제외** | 적용 후 성능 변화가 적고, **운영 효율성** 및 $\text{MVP}$ 단계의 **차원 증가 문제**를 고려하여 최종 제외. | 모델 입력 차원을 줄여 **추론 속도를 최우선**함. |

---

### 3단계: 모델 아키텍처 및 학습 전략 변천사 (시간 순서 명시적 학습)

| 특징 | 내용 | 최종 결과 |
| :--- | :--- | :--- |
| **$\text{Multi-Head Attention}$ 유지** | **$\text{Attention}$과 $\text{PE}$ 둘 다 사용**하는 최종 모델 구조 확정. | $\text{Bi-LSTM}$의 시계열 특징과 $\text{Attention}$의 중요 프레임 집중력을 모두 활용. |
| **$\text{Positional Encoding (PE)}$ 도입** | **$\text{Attention}$과 $\text{LSTM}$만으로는 시간 순서 정의가 어려워** $\text{PE}$를 추가. | **'미국', '무한'**과 같이 **뒷동작**이 중요한 단어의 **시간 순서 관계**를 명시적으로 모델에 제공하여 혼동 문제 개선. |
| **성능 개선** | $\text{PE}$ 도입 후 **'미국', '무한'**과 같은 단어의 초기 예측 정확도 $\mathbf{0\% \sim 20\%}$ 수준이 $\mathbf{40\%}$로 유의미하게 개선됨. | **위치 정보의 명시적 제공**이 미묘한 동작 시퀀스 구분에 효과적임을 입증. |
| **최종 아키텍처** | $\mathbf{\text{1D-CNN}} \to \mathbf{\text{Bi-LSTM}} \to \mathbf{\text{Positional Encoding}} \to \mathbf{\text{Multi-Head Attention}}$ (Change Detector 통합) $\to \mathbf{\text{Classification}}$ | 시간적 특징 학습을 위한 모든 핵심 구성 요소를 통합. |

---

## 개발 과정으로부터 얻은 주요 교훈

1.  **시간 순서 명시의 필요성:** **$\text{LSTM}$과 $\text{Attention}$만으로는** '미국', '무한'처럼 **뒷부분 동작 순서**가 결정적인 수어 단어를 명확히 구분하기 어렵기 때문에, **$\text{Positional Encoding}$**을 통해 **시간적 위치 정보를 명시적으로** 부여하는 것이 필수적임을 확인했습니다.
2.  **효율성 vs. 성능 트레이드오프:** $\mathbf{\Delta X}$ (속도) 특징을 제외하여 **모델의 효율성과 경량화**를 확보하는 동시에, $\text{PE}$를 추가하여 **낮은 비용**으로 **일반화 성능**을 크게 개선하는 균형 잡힌 접근 방식을 택했습니다.
3.  **데이터 전처리의 힘:** $\mathbf{3\text{D}}$ 좌표 복원, **상대 좌표계 변환**, **$\text{Z}$축 평활화**와 같은 수어 특화 특징이 모델 구조 변경보다 **모델의 강건성**을 확보하는 데 훨씬 더 결정적인 역할을 했습니다.
