# SignBell 플랫폼 서비스 선택 동의 약관

본 약관은 SignBell 서비스 이용과 관련하여 **선택적으로 동의**할 수 있는 사항들을 포함합니다.
이러한 동의는 서비스 이용에 필수가 아니며, 동의하지 않아도 기본 서비스 이용이 가능합니다.

- **작성자:** [송민재](https://github.com/songkey06)
- **문서 버전:** v1.0.1

**대상 독자:**
- **운영자/관리자**: 서비스 선택 동의 사항을 기준으로 운영
- **개발자**: 서비스 내 선택 동의 처리 로직 및 기능 구현 시 참고
- **법률 자문가**: 실제 서비스 론칭 전, 법률 검토 및 보완 근거로 활용
- **신규 합류자**: 플랫폼 선택 동의 구조를 빠르게 이해하기 위한 온보딩 자료

---

## **제1장 개인정보 처리**

**제1조 (맞춤형 추천 서비스)**
1. **처리 목적**: 개인화된 학습 콘텐츠 추천, 학습 진도에 맞는 단어 추천
2. **수집 항목**:
    - 학습 패턴 및 선호도 데이터
    - 학습 진도 및 성취도 정보
    - 퀴즈 참여 기록 및 정답률
3. **보유 기간**: 회원 탈퇴 시까지
4. **동의 거부 시**: 일반적인 학습 콘텐츠만 제공됩니다.

**제2조 (마케팅 및 광고 정보 수신)**
1. **처리 목적**: 신규 콘텐츠, 이벤트, 할인 정보 등 마케팅 목적의 알림 제공
2. **수집 항목**:
    - 이메일 주소 (알림 수신용)
    - 휴대폰 번호 (SMS 수신용, 향후 확장 시)
3. **수신 방법**:
    - 이메일 알림
    - 앱 내 푸시 알림 (향후 확장 시)
4. **동의 거부 시**: 마케팅 정보 수신이 제한되며, 서비스 이용에는 영향이 없습니다.

## **제2장 서비스 개선 및 분석**

**제3조 (서비스 이용 분석)**
1. **처리 목적**: 서비스 품질 개선, 사용자 경험 최적화
2. **수집 데이터**:
    - 서비스 이용 패턴 분석
    - 기능별 사용 빈도 통계
    - 오류 및 성능 데이터
3. **데이터 처리**:
    - 개인 식별 정보 제거
    - 집계된 통계 데이터만 활용
4. **동의 거부 시**: 서비스 개선 데이터 제공이 제한되며, 기본 서비스는 정상 이용 가능합니다.

**제4조 (사용자 피드백 수집)**
1. **처리 목적**: 서비스 개선 및 신규 기능 개발
2. **수집 항목**:
    - 사용자 의견 및 제안
    - 서비스 만족도 조사 결과
    - 기능 개선 요청사항
3. **보유 기간**: 피드백 처리 완료 후 1년
4. **동의 거부 시**: 피드백 제공이 제한되며, 서비스 이용에는 영향이 없습니다.

---

## **제3장 AI 모델 성능 개선 데이터 활용 (추가)**

**제5조 (AI 모델 학습 데이터 제공)**
1. **처리 목적**: AI 모델 성능 개선 및 정확도 향상, 서비스 전체 사용자 수어 학습 지원
2. **수집 데이터**:
    - 거울 모드에서 수집된 MediaPipe 좌표 데이터
    - 퀴즈 참여 시 수어 동작 좌표 데이터
3. **데이터 처리**:
    - 개인 식별 정보는 완전히 제거
    - 익명화된 좌표 데이터만 암호화하여 저장 및 AI 학습에 활용
    - 원본 영상은 저장하지 않음
4. **동의 거부 시**: AI 모델의 학습 데이터 제공이 제한되며, **기본 서비스 이용 (AI 피드백 포함)**은 정상적으로 가능합니다.

---

## **제4장 동의 철회 및 변경**

**제6조 (동의 철회)**
1. 언제든지 개별 동의를 철회할 수 있습니다.
2. 동의 철회 시 해당 서비스 이용이 제한될 수 있습니다.
3. 동의 철회는 서비스 설정 또는 고객센터를 통해 가능합니다.

**제7조 (동의 변경)**
1. 동의 범위는 언제든지 변경할 수 있습니다.
2. 변경된 동의는 즉시 적용됩니다.
3. 동의 변경 시 서비스 이용에 미치는 영향을 안내합니다.

## **제6장 약관의 효력**

**제10조 (약관의 효력)**
1. 본 약관은 선택적 동의 사항으로, 동의하지 않아도 기본 서비스 이용이 가능합니다.
